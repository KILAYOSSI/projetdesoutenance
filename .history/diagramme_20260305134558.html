Rôles des Utilisateurs

Cardinalités

Rôle Description Permissions
VISITEUR Utilisateur non connecté Parcourir produits, recherches
ACHETEUR Utilisateur connecté et vérifié Acheter, passer commandes, voir panier
PRODUCTEUR Acheteur + KYC validé + Publier produits, gérer exploitations
ADMIN Administrateur + Valider KYC, gérer plateforme
Relation Cardinalité Explication
Utilisateur → Kyc 1 : 0..* Un utilisateur peut soumettre 0 ou plusieurs demandes KYC
Utilisateur → Produit 1 : 0..* Un producteur peut publier 0 ou plusieurs produits
Utilisateur → Commande 1 : 0..* Un client peut passer 0 ou plusieurs commandes
Utilisateur → Exploitation 1 : 0..* Un agriculteur peut avoir 0 ou plusieurs exploitations
Commande → LigneCommande 1 : 0..* Une commande contient 0 ou plusieurs lignes
Commande → Paiement 1 : 0..1 Une commande peut avoir 0 ou 1 paiement
Exploitation → SuiviCulture 1 : 0..* Une exploitation peut avoir 0 ou plusieurs cultures
Categorie → Produit 1 : 0..* Une catégorie peut contenir 0 ou plusieurs produits
